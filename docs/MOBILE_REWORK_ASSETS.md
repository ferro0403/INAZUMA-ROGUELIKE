# Mobile rework assets

This mobile rework does not add third-party raster assets. The stadium, tunnel, pitch, rarity glow, inventory, timeline, modal, victory and defeat atmospheres are generated with local CSS gradients, masks, shadows and pseudo-elements in `css/mobile-rework.css`, `css/mobile-components.css`, and `css/mobile-screens.css`.

## External sources

None.

## Licenses

No new external asset license is required. Existing player portraits, fullbody images, team logos, local database records and the season-aware visual resolver remain unchanged and continue to be sourced from the repository's existing assets/data.
